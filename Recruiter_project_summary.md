# Recruiter Project Summary

## Project Name

Accounts Payable Aging Report Automation using SQL and n8n

## Objective

To automate daily AP aging reporting by identifying outstanding vendor invoices, calculating overdue days, assigning aging buckets, and emailing the report to finance stakeholders.

## Business Impact

* Reduced manual Excel reporting
* Improved visibility of overdue vendor invoices
* Helped finance team prioritize payments
* Supported month-end close and cash-flow planning
* Strengthened AP controls

## Technical Skills Demonstrated

* SQL reporting
* Accounts Payable process knowledge
* AP aging analysis
* Workflow automation using n8n
* Automated email reporting
* Data validation and finance controls

## Project Highlights

### Automation Benefits
- **Time Savings**: Eliminated daily manual Excel report creation
- **Data Accuracy**: Automated calculations reduce human errors
- **Timeliness**: Reports delivered automatically at 9 AM daily
- **Scalability**: Works with any volume of invoices

### Key Deliverables
- Automated n8n workflow with 4 integrated nodes
- MySQL database with vendor and invoice tables
- Daily CSV report generation
- Automated email distribution to finance team
- 5 SQL scripts for various reporting needs

### Technologies & Tools
- **Database**: MySQL
- **Automation**: n8n (4 connected nodes)
- **Languages**: SQL, JSON
- **Output Format**: CSV, HTML Email
- **Scheduling**: Daily at 9:00 AM
